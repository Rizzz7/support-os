Troubleshoot errors while you edit Google Docs, Sheets, Slides, & Vids
If you get “An error occurred” which prevents you from making edits on your Docs, Slides, Sheets, or Vids, you can troubleshoot the issue. If you urgently need to access a file and resume your progress, you can follow these steps to resume your workflow.

"Encountered an error. Please try reloading this page, or coming back to it in a few minutes."
"A server error occurred. Please press Reload in your browser."
If you get these messages, wait for 5 minutes then reload the page.

# Gmail Troubleshooting Guide

This file is a project-ready retrieval sheet for common Gmail issues, error messages, and practical recovery steps. It is based on Google’s Gmail help documentation for common Gmail errors. [page:3]

## How to use this guide

Start by identifying the symptom: sign-in failure, no connection, browser error, attachment problem, sync issue, spam issue, or account security concern. Then follow the matching fix sequence below from easiest to more invasive. [page:3]

## Quick triage workflow

1. Check whether Gmail is having an outage.
2. Test the issue in another browser or device.
3. Clear cache, cookies, or app data if the problem looks browser-related.
4. Retry login, sync, or sending after the basic checks. [page:3]

## Common issues and fixes

### 1. Temporary Error (502)
**Symptoms:** Gmail shows a temporary error on sign-in.  
**Likely cause:** Gmail is temporarily unavailable.  
**Fix steps:**
- Wait a few minutes and try again.
- Confirm whether Gmail is having an outage.
- Retry sign-in after the error clears. [page:3]

### 2. No connection
**Symptoms:** Gmail says there is no connection.  
**Likely cause:** Internet, device, or network connectivity problems.  
**Fix steps:**
- Check your internet connection.
- Restart the device.
- Turn Wi‑Fi or mobile data off and back on.
- Try another network.
- If needed, contact your ISP or network host. [page:3]

### 3. Gmail won’t open in browser
**Symptoms:** Gmail fails to load normally.  
**Likely cause:** Browser issues or browser compatibility problems.  
**Fix steps:**
- Close and reopen the browser.
- Update the browser.
- Try a different supported browser. [page:3]

### 4. Gmail app won’t open
**Symptoms:** The mobile app is stuck or fails to launch.  
**Likely cause:** App session or device-side issue.  
**Fix steps:**
- Close and reopen the Gmail app.
- Restart the phone.
- Check whether the app needs an update. [page:3]

### 5. Oops or Bad Request
**Symptoms:** Gmail displays `Oops` or malformed request errors.  
**Likely cause:** Browser extensions, add-ons, or corrupted browser state.  
**Fix steps:**
- Disable extensions one by one.
- Try incognito or private browsing mode.
- Clear cache and cookies.
- Reopen Gmail and test again. [page:3]

### 6. Browser extension conflict
**Symptoms:** Gmail behaves oddly only in one browser.  
**Likely cause:** An extension is interfering.  
**Fix steps:**
- Turn off extensions temporarily.
- Remove suspicious add-ons.
- Test Gmail again after each change. [page:3]

### 7. Cache and cookies issue
**Symptoms:** Gmail loads incorrectly, loops, or misbehaves after sign-in.  
**Likely cause:** Corrupted browser data.  
**Fix steps:**
- Clear browser cache and cookies.
- Sign in again.
- Recheck whether the issue persists across sessions. [page:3]

### 8. Advanced setting issue
**Symptoms:** Gmail errors continue after basic fixes.  
**Likely cause:** An advanced feature may be causing the problem.  
**Fix steps:**
- Open Gmail settings.
- Go to the **Advanced** tab.
- Disable enabled advanced features one by one.
- Save changes and retest. [page:3]

### 9. Email not sending
**Symptoms:** Message does not go out.  
**Likely cause:** Connectivity, browser, or temporary Gmail issue.  
**Fix steps:**
- Check internet.
- Refresh Gmail or reopen the app.
- Retry after temporary errors clear. [page:3]

### 10. Email stuck in outbox
**Symptoms:** Message remains queued.  
**Likely cause:** Weak connection or app sync issue.  
**Fix steps:**
- Reconnect to a stronger network.
- Reopen Gmail.
- Try sending again after reconnecting. [page:3]

### 11. Attachment problems
**Symptoms:** Attachment won’t send or does not appear.  
**Likely cause:** File size or display issue.  
**Fix steps:**
- Check whether the attachment is too large.
- Reopen long email threads to reveal hidden attachment areas.
- Use Google Drive for large files. [page:3]

### 12. File size limit exceeded
**Symptoms:** Gmail rejects a large attachment.  
**Likely cause:** Email attachment size is too large.  
**Fix steps:**
- Keep attachments under Gmail’s limit.
- Move large files to Google Drive and share a link. [page:3]

### 13. Missing email in inbox
**Symptoms:** You expect a message but can’t find it.  
**Likely cause:** It may be in Spam, Trash, filtered, or archived.  
**Fix steps:**
- Search using sender name or keywords.
- Check Spam and Trash.
- Review filters and rules. [page:3]

### 14. Email lost after sending
**Symptoms:** A sent or received email seems missing.  
**Likely cause:** Search filters, labels, or folder routing.  
**Fix steps:**
- Search broadly for the email.
- Check all folders and labels.
- Review automatic filters. [page:3]

### 15. Forgot password
**Symptoms:** You cannot sign in.  
**Likely cause:** Password is forgotten or incorrect.  
**Fix steps:**
- Use the account recovery flow.
- Reset the password through Google’s recovery process.
- Confirm you are using the correct email address. [page:3]

### 16. Wrong email address entered
**Symptoms:** Sign-in keeps failing even though the password seems right.  
**Likely cause:** Wrong account identifier.  
**Fix steps:**
- Re-enter the full email address carefully.
- Check for typos and alternate accounts.
- Try another browser if autofill is confusing the entry. [page:3]

### 17. Spam folder problem
**Symptoms:** Expected mail lands in Spam or your own messages get flagged.  
**Likely cause:** Message content, filters, or reputation signals.  
**Fix steps:**
- Review spam folder regularly.
- Mark legitimate mail as Not Spam.
- Avoid suspicious links and phrasing in outgoing mail. [page:3]

### 18. Notifications not working
**Symptoms:** New mail alerts do not appear.  
**Likely cause:** Notifications are disabled in Gmail or device settings.  
**Fix steps:**
- Enable notifications in Gmail.
- Check phone or browser notification permissions.
- Retest after saving the permission settings. [page:3]

### 19. Links do not open
**Symptoms:** Clicking a Gmail link goes nowhere.  
**Likely cause:** Browser or default-app issue.  
**Fix steps:**
- Set the correct default browser.
- Clear browser cache and cookies.
- Try another browser. [page:3]

### 20. Gmail sync issue on mobile
**Symptoms:** Messages do not sync on the phone.  
**Likely cause:** App cache, app version, or connectivity problem.  
**Fix steps:**
- Check the internet connection.
- Update the Gmail app.
- Clear the app’s cache.
- Re-sync the account. [page:3]

### 21. Outage confusion
**Symptoms:** Gmail fails in many places at once.  
**Likely cause:** Service outage rather than your device.  
**Fix steps:**
- Check the Google Workspace Status Dashboard.
- Wait for the outage to resolve.
- Retry later rather than changing many settings at once. [page:3]

### 22. Mobile sign-out limitation
**Symptoms:** You want to sign out of Gmail on Android/iPhone.  
**Likely cause:** Mobile Gmail app does not allow direct sign-out the same way desktop does.  
**Fix steps:**
- Remove the Google Account from the device if needed.
- Add it back after troubleshooting. [page:3]

### 23. Browser version incompatibility
**Symptoms:** Gmail works inconsistently after a browser update or on an old browser.  
**Likely cause:** Unsupported or outdated browser version.  
**Fix steps:**
- Update the browser.
- Switch to another supported browser. [page:3]

### 24. Security concern or hacked account
**Symptoms:** Suspicious login activity, unknown devices, or unexpected changes.  
**Likely cause:** Account compromise.  
**Fix steps:**
- Change your password immediately.
- Review security activity and devices.
- Enable two-factor authentication. [page:3]

### 25. Storage full
**Symptoms:** Gmail slows down, stops receiving messages, or sync feels broken.  
**Likely cause:** Google account storage is full.  
**Fix steps:**
- Delete unnecessary Drive files.
- Remove old large emails.
- Search for large attachments and clean them up. [page:3]

### 26. Slow loading
**Symptoms:** Gmail takes too long to open.  
**Likely cause:** Browser load, too many tabs, or slow internet.  
**Fix steps:**
- Update the browser.
- Close extra tabs.
- Check connection speed. [page:3]

### 27. Message formatting or display looks broken
**Symptoms:** Gmail UI looks incorrect or elements do not render well.  
**Likely cause:** Browser state or extension interference.  
**Fix steps:**
- Test in incognito mode.
- Disable extensions.
- Clear cache and cookies. [page:3]

### 28. One device works, another doesn’t
**Symptoms:** Gmail works on one device but not another.  
**Likely cause:** Device-specific browser or app state.  
**Fix steps:**
- Compare browser version and extensions.
- Clear the affected device’s cache.
- Update the Gmail app or browser. [page:3]

### 29. Gmail keeps asking to sign in
**Symptoms:** Sessions do not stay logged in.  
**Likely cause:** Cookies disabled or browser data clearing.  
**Fix steps:**
- Enable cookies.
- Check privacy extensions.
- Avoid auto-clearing session data if you want persistent sign-in. [page:3]

### 30. Need a general recovery path
**Symptoms:** The issue does not match any single known problem.  
**Likely cause:** Most Gmail issues are caused by connectivity, browser state, extensions, app state, or outages.  
**Fix steps:**
- Check connection.
- Check outage status.
- Try another browser or device.
- Clear cache/cookies.
- Test again after restarting the app or device. [page:3]

## Troubleshooting checklist

Before escalating, confirm the following:
- Internet connection is stable.
- Gmail is not experiencing a service outage.
- Browser is updated and supported.
- Extensions are not interfering.
- Cache and cookies have been cleared.
- Mobile app is updated and synced. [page:3]

## Best practices

Keep Gmail maintenance simple: update browsers and apps, use incognito mode to isolate extension problems, and regularly clean storage if mail volume is high. For security-sensitive accounts, prioritize password recovery, suspicious login review, and two-factor authentication. [page:3]
